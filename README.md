## Como rodar este projeto (sem Docker)

Pré‑requisitos:
- Node.js (versão LTS)
- npm

Passos:

```sh
# Instalar dependências
npm install

# Rodar em modo desenvolvimento
npm run dev
# (ou se existir apenas start:)
# npm start
```

Abra em: http://localhost:3000

### Scripts úteis
```sh
npm run build   # Gera build de produção
npm run lint    # Verifica lint
```
